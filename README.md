# Compact presets for OpenIPC FPV devices

This research will be conducted in collaboration with the laboratory of [University of Guglielmo Marconi][university], Rome, Italy, and aims to create a system and standardize presets for all types of OpenIPC FPV firmware on Goke, Hisilicon, Rockchip, SigmaStar, and other processors.
